# 🤖 Discord Bot Project

บอท Discord เขียนด้วย Python ใช้งานง่าย ปรับแต่งได้ตามใจ!

## 🚀 ฟีเจอร์
- ส่งข้อความอัตโนมัติ
- รองรับคำสั่ง custom
- ใช้ Discord API ผ่าน `discord.py`
- เขียนโค้ดสะอาด เข้าใจง่าย
